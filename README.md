📱 Responsive Webpage Using Media Queries

A fully responsive webpage built using HTML5, CSS3, Flexbox, and Media Queries. The project demonstrates how a desktop-first layout can be adapted for tablet and mobile screens using proper breakpoints.

🚀 Project Overview

This project includes:

Navigation Bar with two menus

Hero Section with left image and right text

Product Section with multiple product cards

Footer with social media icons

Fully responsive layout using media queries (576px & 768px breakpoints)

The design automatically adjusts layouts (flex direction, spacing, image sizes, etc.) for mobile and tablet screens.
